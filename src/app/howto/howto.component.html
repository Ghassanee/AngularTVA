How to use this Application ?

Interface d Acceuil

Cette interface contient trois elements :

- Liste de societe:

cet element contient la partie creation d une nouvelle societe si n existe pas deja dans la base de donnes ou modifier une societe si elle existe .
- How To Use this App : Ci celui -ci.
- Start the Application : Cet element vous permet de commencer a utiliser l application :


- 1er Etape: Choisir un choix :

. Si la societe existe dans la base de donnees.
. Si la societe n existe pas dans la base de donnees.


- 2eme Etape :

. Si la societe existe -> On va choisir la societe dans un "select " qui affiche tous les societes.
. On va diriger vers la page de creation d une societe avec ces employes .


- 3eme Etape : Choisir l impot a payer

. Declaration TVA -> Diriger vers la page de paymment de TVA
. Declaration IS -> Diriger vers la page de paymment de IS
. Declaration IR -> Diriger vers la page de paymment de IR
